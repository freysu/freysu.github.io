使用 rem 单位设置移动端尺寸

移动端的特点

  1. 设备尺寸不同

  2. 可以通过设置百分比解决

  3. 设置百分比计算繁琐

单位概述

  1. px：

    像素

  2. em:

    相对于父级的 font-size 值,父级的 font-size 值越大，子级的 em 就越大

  3. rem:

相对于 html 标签的 font-size 值
html font-size: N px; 1rem 代表 N px; 10rem 代表 10N px;
