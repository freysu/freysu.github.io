常用的 html 标签以及属性

^a7d7b1

  1. div 标签：

    无语义标签，划分整体、多个标签的区域

  2. span 标签：

    无语义标签，可以把一行文字划分成多个区域

  3. h1 ~ h6 标签：标题标签

  4. p 标签：段落标签

  5. ul 标签：无序列表

    列表元素: li 配合使用

    标题标签最好不要放到列表里面，放到列表里面会缩进

  6. a 标签：

    href 属性：可以指定跳转的地址

  7. img 标签：图片标签

    src 属性：图片地址

    alt 属性：当图片没加载成功就会显示出来

    title 属性：给图片命名

  8. 通过 id 属性给标签定一个唯一值。类似名字

  9. class 属性分类，分类之后可以为标签指定不同的样式
