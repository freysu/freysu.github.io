申请百度翻译 API 教程

1. 首先到百度翻译开放平台首页(http://api.fanyi.baidu.com)点击右上角进行登录。

2. 登录后，选择 “管理控制台”。

3. 首次进入 “管理控制台”，需要首先注册为百度翻译的开发者身份。在这里，我们选择 “个人开发者”，并填写相关信息。（这一步很重要！）

  • 注册成功后，弹出一个提示进行身份认证的窗口；依据大家的需要选择是否要进行身份认证。建议大家还是选择 “确定”，因为可以看一下认证与否对于我们翻译接口的影响。

  • 因为实名认证之后可以升级到高级版。标准版和高级版的调用频率有区别，所以会出现多条翻译结果出不来的情况，为了提高体验，如果方便可以尽可能地进行实名认证升级成高级版！

    如下图，可以看到如果认证了个人身份，即可使用高级版的翻译接口。

4. 随后，在这一界面点击上方 “翻译开放平台” 按钮，回到主页。

可以看到，此时 “管理控制台” 中已经有了我们账户的信息，同时接口 API 以及密钥已经显示在了左下角红色圈内部分。

5. 点击上图中粉色框内的 “立即开通”。

点击上图中粉色框内的 “立即开通”。（这一步非常重要）

点击上图中粉色框内的 “立即开通”。（这一步非常重要）

（重要的事情说三遍）

6. 选择【通用翻译】。

7. 在这里，我选择标准版，大家如果需要高级版就选择右侧即可。

8. 不管是标准版还是高级版都不要填服务器地址！（千万要留意），其他信息随便填。

不管是标准版还是高级版都不要填服务器地址！

不管是标准版还是高级版都不要填服务器地址！

（重要的事情说三遍）

9. 随后，回到首页，即可看到我们刚刚申请的通用翻译服务已开通了。

10. 最后一步，在论文翻译降重助手中配置账号。

上图的红框部分就是我们刚刚申请的 APP ID 和密钥。

我们需要将填写到论文翻译降重助手(https://freysu.github.io/MyTranslator)的 appid 输入框。同理，我们还要把刚刚申请的密钥填写到 key 输入框。

最后在论文翻译降重助手那里点击保存即可，就可以翻译降重啦。

如果还有疑问，欢迎进群反馈交流。 QQ 交流群（群号：238223706）

常见问题 QA：

  1. Q：如果出现配置账号出错的情况
    A：请先检查通用翻译服务的开通情况，如果开通了再检查是否填错**百度翻译 API 的 key 和 appid **

  2. Q：如果出现 翻译的时候出现很多翻译结果不出来(比如 6 个结果里只出来 1 个或 2 个) 的情况

    A：请留意一下自己申请的 api 是哪一个版本。 标准版可能会发生以上情况。高级版大概率是不会的。
    实名认证之后可以免费申请高级版。

  3. Q：如果出现配置账号正确但不出翻译结果的情况
    A：请检查是否填写了服务器地址。
