ECMAScipt

ECMAScipt 简称 ES。

简单的理解就是用 ES 表示 JS 的版本。

例如，Java8、Java9，说的就是 Java SE 的版本；Python2、Python3，说的就是 Python 的版本。

ES2015(ES6)是一个跨越性很大的一个版本。

ES2015 基础语法

变量

  1. 使用 let代替 var。

  2. 块级作用域。

  3. 不存在变量提升。

    var存在变量提升。

  4. 不允许重复声明。

总之，让变量更加规范。

常量

  1. const定义常量。

  2. 定义之后不可以修改。

  3. 不变的值用常量声明。

  4. 函数表达式可以使用常量。

    函数表达式常用const来定义。

  5. 对象声明可以使用常量。

  6. 引入外部模块使用常量。

模板字符串

反引号 ``

  1. 支持换行。
  2. 支持嵌入变量。
    ${}
    连接字符串

解构赋值

  1. 数组的解构赋值。
    [n,m]

  • 例子 - 交换。让 n=20,m=10。
    定义一个临时变量，先把 n 放到 temp 里，再把 m 赋值给 n,最后再把 temp 赋值给 n。
