按条件执行

条件运算符

条件（三元）运算符是 JavaScript 仅有的使用三个操作数的运算符。一个条件后面会跟一个问号（?），如果条件为 truthy，则问号后面的表达式 A 将会执行；表达式 A 后面跟着一个冒号（:），如果条件为falsy，则冒号后面的表达式 B 将会执行。本运算符经常作为 if 语句的简捷形式来使用。

boolean ? value1 : value2

switch 语句

分支语句

可以实现多重选择，某些情况相对于 if 语句更简洁。

switch (表达式) {

case 值1: 语句1; break;

case 值2: 语句2; break;

case 值3: 语句3; break;

default: 语句4;

}
